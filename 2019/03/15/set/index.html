Set

ES6 提供的数据结构类似于数组，值是唯一的，通过 Set 构造函数来生成

属性和方法

weakSet

和Set 类似 但是只能存储对象，不可遍历
如果其他对象未应用该对象，垃圾回收机制会自动收回所占用的内存
适合存储临时对象和绑定对应对象的信息

由于 WeakSet 的成员都是弱引用随时可能被回收，因此可以存储DOM节点，不用担心节点移除时，发生内存泄漏。
